## Требования

- **Python 3.6 или выше** (только если запускать из исходного кода).
- Дополнительные библиотеки не требуются – используется только стандартная библиотека (`tkinter` входит в состав Python).

Как пользоваться
1. Введите ранее накопленное время в формате ЧЧ:ММ (например, 05:30).

2. Введите время начала совещания (ЧЧ:ММ).

3. Введите время окончания совещания (ЧЧ:ММ).
   Если совещание заканчивается после полуночи (например, 23:30 – 01:15), это корректно учитывается.

4. Нажмите кнопку Рассчитать (или клавишу Enter).
  В поле результата появятся:

    - старое накопленное время;

    - начало и конец совещания;

    - длительность совещания;

    - новое накопленное время.

5. Чтобы скопировать результат, выделите текст и нажмите Ctrl+C или щёлкните правой кнопкой мыши и выберите Копировать.

6. Кнопка Сбросить очищает все поля для нового расчёта.
