# Meeting Calculator

Простое приложение с графическим интерфейсом для расчёта длительности совещания и обновления накопленного времени. Программа учитывает переход через полночь (если совещание заканчивается после полуночи) и позволяет скопировать результат в буфер обмена.

## Возможности

- Расчёт длительности совещания по времени начала и окончания
- Автоматический учёт перехода через 00:00 (например, начало 23:00, окончание 01:00)
- Прибавление длительности к накопленному времени
- Поддержка формата времени ЧЧ:ММ (часы и минуты)
- Запрет редактирования результата, но возможность выделения и копирования текста (правой кнопкой мыши или Ctrl+C)
- Сброс всех полей одной кнопкой

## Требования

- **Python** версии 3.6 или выше
- Стандартная библиотека Python (tkinter входит в поставку)

## Установка и запуск

### Запуск скрипта

1. Убедитесь, что Python установлен и добавлен в `PATH`.
2. Дважды кликните на файл
